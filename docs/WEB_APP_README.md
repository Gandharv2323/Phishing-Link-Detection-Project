## ğŸ› Troubleshooting

### "Model not loaded"
**Solution:** Run `python hsef_model.py` first

### Port already in use
**Solution:** Change port in `app.py` or:
```bash
# Kill process on port 5000
# Windows:
netstat -ano | findstr :5000
taskkill /PID <PID> /F

# Linux/Mac:
lsof -ti:5000 | xargs kill -9
```

### Slow predictions
**Solution:** 
- Check GPU availability
- Reduce batch size
- Use caching
- Enable fast_mode in training
